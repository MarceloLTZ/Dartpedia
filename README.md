Descrição

O Dartpedia é um aplicativo de linha de comando (CLI) desenvolvido em Dart durante a disciplina. 
O objetivo do projeto é permitir a execução de comandos básicos diretamente pelo terminal, aplicando conceitos de programação, organização de código e controle de versão com Git e GitHub utilizando o Google Cloud Shell.

Como executar

No terminal, execute:

dart run bin/dartpedia.dart
Comandos disponíveis
Help

Exibe a lista de comandos disponíveis:

dart run bin/dartpedia.dart help
Version

Exibe a versão atual do aplicativo:

dart run bin/dartpedia.dart version
Search

Estrutura inicial para busca:

dart run bin/dartpedia.dart search
